Calculo de ISR – Republica Dominicana

Este programa en C# calcula el Impuesto Sobre la Renta (ISR) según la escala salarial vigente de la DGII.

El sistema solicita el sueldo del empleado y determina si debe pagar impuesto o si esta exento, mostrando el resultado en pantalla.

Ejecucion del programa

Ejecutar el proyecto en Visual Studio.

Ingresar el sueldo solicitado.

El sistema mostrará el ISR a pagar o "N/A" si no aplica.
